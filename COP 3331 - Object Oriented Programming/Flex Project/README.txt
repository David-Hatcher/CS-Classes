The Disappearance of Emily Jane

Run Instructions:

Upload the entire zip file to the student cluster. Navigate to the folder you uploaded it in. Run the following commands in order:

module purge

module add compilers/gcc/7.3.0

upzip FlexProject.zip

cd DEJ

g++ *.cpp

./a.out


Playing instructions:

Follow the on screen prompts. You can access the available commands in game by typing "commands".

text in brackets (ie [ drawer ]) can be interacted with using one or more of the following commands:

touch, look, check, open, break, read

go move to a different room type go room_name(ie "go library")

to quit the game type exit, this will save your progress
